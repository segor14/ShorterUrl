# UrlShorter

Сервис сокращения ссылок с авторизацией пользователей.

## Стек технологий
- **FastAPI**: Веб-фреймворк.
- **PostgreSQL**: База данных.
- **SQLAlchemy (Async)**: ORM.
- **uv**: Менеджер пакетов и зависимостей.
- **Docker & Docker Compose**: Контейнеризация.

## Особенности
- Регистрация и аутентификация пользователей (JWT).
- Использование расширения `pgcrypto` в PostgreSQL для безопасного хеширования паролей на стороне БД.
- Автоматическое создание таблиц при запуске приложения.

## Запуск проекта

### С использованием Docker Compose (рекомендуется)

1. Убедитесь, что у вас установлен Docker и Docker Compose.
2. Склонируйте репозиторий.
3. Запустите проект:
   ```bash
   docker-compose up --build
   ```
4. Приложение будет доступно по адресу: `http://localhost:8000`
5. Документация API (Swagger): `http://localhost:8000/docs`

### Локальный запуск (без Docker)

1. Установите `uv`:
   ```bash
   curl -LsSf https://astral.sh/uv/install.sh | sh
   ```
2. Установите зависимости:
   ```bash
   uv sync
   ```
3. Настройте переменные окружения в файле `.env` (см. `src/settings.py` для параметров по умолчанию). Обязательно наличие работающего PostgreSQL с расширением `pgcrypto`.
4. Запустите приложение:
   ```bash
   uv run uvicorn main:app --reload
   ```

## API Эндпоинты

### Аутентификация
- `POST /auth/register`: Регистрация нового пользователя.
- `POST /auth/login`: Получение JWT токена.

### Пользователи
- `GET /users/me`: Получение информации о текущем пользователе.

### Ссылки
- `POST /links/shorten`: Создать короткую ссылку. Можно передать `custom_alias` и `expires_at`. (Требуется авторизация).
- `GET /links/{short_code}`: Редирект на оригинальный URL.
- `GET /links/{short_code}/stats`: Статистика по ссылке.
- `GET /links/search?original_url={url}`: Поиск короткой ссылки по полному URL.
- `PUT /links/{short_code}`: Обновить оригинальный URL для короткого кода (Требуется авторизация, только для владельца).
- `DELETE /links/{short_code}`: Удалить короткую ссылку (Требуется авторизация, только для владельца).
